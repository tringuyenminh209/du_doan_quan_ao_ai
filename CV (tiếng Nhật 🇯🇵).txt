【プロジェクト名】AIによるファッション画像分類Webアプリ

【使用技術】Python / TensorFlow / Keras / Flask / HTML / CSS / GitHub

【概要】
Fashion-MNISTデータセットを使用して、CNNモデルを構築し、服の画像を自動で分類するAIシステムを開発しました。Flaskを使ってWebアプリとして実装し、画像アップロード・予測・結果表示をすべてリアルタイムで行えるようにしました。

【特徴】
- AIモデルを自作して学習・保存（model.h5）
- FlaskによるWebアプリ化（HTML/CSS含む）
- ダークモード・多言語対応（日本語・英語・ベトナム語）
- GitHubでソースコード管理・公開
